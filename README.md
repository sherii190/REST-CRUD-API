## Utveckla

**Kon igång**

*Clone repot med länken nedan:*

```
git clone https://github.com/sherii190/REST-CRUD-API.git

REST-CRUD-API
```

*Installera följande saker efteråt:*

```
npm i express 
```

```
npm i nodemon 
```

```
npm i uuid4 
```

* Kör programmet:*
```
npm start 
```
